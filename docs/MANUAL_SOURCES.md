## 1. Топ-приоритет — проверять ежедневно (≈ 5–7 мин/день)

### 1.1 LinkedIn Job Alerts

**Почему ценен:** крупнейший пул вакансий, многие компании размещают _только_ здесь;
особенно полезен для ops/support-ролей, которые не публикуются на крипто-бордах.
Bybit недавно постил там «CIS Social Media Specialist» — не было нигде больше.

**Почему вручную:** автоматический скрейпинг нарушает ToS LinkedIn; VPS-IP блокируется.
Алерты на email — официальный и разрешённый путь.

**Ритм:** ежедневно — алерты сами приходят на почту.

**Время:** 2–3 мин на просмотр письма.

**Лайфхак — настрой один раз:**
1. linkedin.com/jobs → в строке поиска вбивай по очереди:
   - `"crypto operations" remote`
   - `"web3 operations" remote`
   - `"DeFi operations"` (без привязки к стране)
   - `"web3 support" Russian` (CIS-команды)
   - `"AI automation specialist" remote`
2. После каждого запроса → нажми **«Создать уведомление»** → частота: ежедневно.
3. Письма будут приходить на sanny3606@ya.ru — достаточно просматривать
   заголовки и открывать интересные.

## 2. Средний приоритет — 2–3 раза в неделю (≈ 15–20 мин/неделю)

### 2.1 cryptocurrencyjobs.co

**Где:** https://cryptocurrencyjobs.co

**Почему ценен:** один из самых авторитетных крипто-бордов, активен в 2026 (72 новых вакансии
за июль на момент аудита). Многие компании публикуют туда эксклюзивно, не через Greenhouse/Ashby.
Фокус на non-dev роли: ops, finance, legal, community.

**Почему вручную:** HTML-only, нет публичного API; VPS-риск для скрейпинга неизвестен
(с личного браузера работает без ограничений).

**Ритм:** 2–3 раза в неделю.

**Время:** 5 мин.

**Лайфхак:** На сайте есть фильтр по категориям — выбирай **Operations** и **Customer Support**.
Есть RSS-фид: добавь в любой RSS-ридер (например, Feedly) для пассивного мониторинга.

---

### 2.2 crypto.jobs

**Где:** https://crypto.jobs/jobs

**Почему ценен:** 3 500+ активных вакансий в 2026, включая non-dev и ops-роли.
Часть позиций не пересекается с другими бордами.

**Почему вручную:** HTML-скрейп, VPS-риск неизвестен; с браузера открывается без проблем.

**Ритм:** 2 раза в неделю.

**Время:** 5 мин.

**Лайфхак:** Используй фильтр **Role Type → Operations / Support** + **Location → Remote**.
Результат сохрани как закладку с параметрами фильтра.

### 2.3 Habr Career — блокчейн и крипто

**Где:** https://career.habr.com/vacancies?s=1&specialization=crypto

**Почему ценен:** главная RU IT-карьерная площадка после HH. Здесь бывают
крипто-ops и support-позиции от CIS-компаний, которые не размещают объявления на международных
бордах. Особенно актуален для AI automation (компании ищут через Habr, не через крипто-каналы).

**Почему вручную:** API требует OAuth 2.0 и регистрации приложения через Habr —
средние трудозатраты на автоматизацию; ручной просмотр гораздо быстрее.

**Ритм:** 2 раза в неделю.

**Время:** 5 мин.

**Лайфхак:** Зарегистрируйся и настрой email-уведомление на Habr Career для ключевых слов
«крипто», «blockchain», «web3», «операции» — бесплатно, приходит на почту.

### 2.4 Bybit Careers

**Где:** https://www.bybit.com/en/promo/global/careers

**Почему ценен:** один из крупнейших мировых криптобирж (~70 млн пользователей),
не покрытый ни одним из наших ATS-парсеров. Регулярно открывает operations, support
и CIS-специализированные роли (пример: «CIS Social Media Specialist» в Дубае).

**Почему вручную:** кастомная карьерная страница без стандартного ATS-API.

**Ритм:** 2–3 раза в неделю (пополняется активно).

**Время:** 5 мин.

**Что смотреть:** поиск по «operations», «support», «CIS», «community». Страница
может быть медленной — дождись полной загрузки. Зарегистрированные кандидаты
могут подписаться на email job alerts через кнопку на сайте.
